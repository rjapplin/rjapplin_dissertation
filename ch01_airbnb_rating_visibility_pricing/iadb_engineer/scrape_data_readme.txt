Scraping New Data

1. Open the inside_air_bnb.Rproj in R (RStudio recommended).
2. Open the R script 00_setupEnv.R
3. Source the script. If any errors pop because of uninstalled packages, use install.packages() to install any missing packages.
4. If not installed on your computer, install Firefox browser. 
5. In the console, type scrapeIAWrapper()

Everything else takes care of itself.
