This directory contains the code necessary to produce the results that appear in Chapter 1 of *Applin (2025).* In addition, it includes a database file containing processed results—but no raw data.

**Bibliography**

R. Jacob Applin, "Rating Visibility and Pricing in Airbnb Markets," in *Applied Essays in Industrial Organization: Pricing in Short-Term Rental Markets and Policy Implications in Tobacco Consumption* (PhD diss., University of Kentucky, 2025).
